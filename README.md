# 📁 Portfolio

To moje proste portfolio stworzone z użyciem **Node.js** i **EJS**.  
Zawiera stronę z informacjami o mnie, listę projektów oraz formularz kontaktowy, który wysyła dane do **Google Docs** (przez Google Forms).

## 🔧 Technologie

- Node.js  
- Express.js  
- EJS (Embedded JavaScript Templates)  
- Google Forms (formularz kontaktowy)  
- HTML, CSS

## 📝 Funkcje

- Wyświetlanie projektów z pliku JSON  
- Sekcja "O mnie"  
- Formularz kontaktowy z integracją Google Forms  
- Prosty, responsywny layout
